#Point I should remember
Math.ceil(n);
	returns the min integer which is <= n.
 
ceil(1.5); returns 2
ceil(-0.3) returns 0
ceil(-1.4); returns -1

Math.floor(n);
	returns the max integer that is >= n;

floor(3.4); returns 3;
floor(-5.06); returns -6;


Math.sqrt(n);
	returns the square root of n;

	sqrt(4); returns 2;
	sqrt(9); returns 3;
